WeatherPy: Latitude and Weather


WeatherPy is built to analyze and visualize the correlation between temprature and other factors when we move from the equator.

To create this project, we enlisted the help of the CitiPy Python library find unique cities all across the globe. To make sure we received a decent amount of cities, we created a loop to run through our citipy library and ensure a spread of 500 different cities.

After recieving our cities, we ran the collected cities through OpenWeatherMap API to build out data surrounding the specific data points we want to compare. This gave us the all the data necessary for our project.

To tie everything together in a beautiful package, we use matplotlib to visualize the data points in scatterplot graphs (shown above).
